# 🚗 Gryphon Rental

Gryphon Rental is a **full-stack Car and Bike Rental System** designed to make renting vehicles **easy, fast, and secure**. It offers a responsive frontend, a robust backend, and efficient database management to provide a smooth experience for both customers and rental service providers.


## ✨ Features

- **User Registration & Authentication** – Secure sign-up/login system.
- **Vehicle Listings** – Browse available cars and bikes with details.
- **Booking System** – Reserve vehicles online with real-time availability.
- **Payment Integration** – Secure and easy online payments.
- **Responsive Design** – Works across desktops, tablets, and mobiles.
- **Admin Dashboard** – Manage vehicles, bookings, and users.


## 🛠 Tech Stack

**Frontend:** HTML, CSS, JavaScript, React (optional)  
**Backend:** Python (Django/Flask) or Node.js (depending on your build)  
**Database:** MySQL / PostgreSQL / SQLite  
**Authentication:** JWT / Django Auth System  
**Payment Gateway:** Razorpay / Stripe (optional)
